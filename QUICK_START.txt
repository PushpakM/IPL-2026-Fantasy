STEP 1: CREATE YOUR PROJECT FOLDER
─────────────────────────────────────
Open Terminal/Command Prompt and run:

  mkdir ~/IPL2026_Fantasy && cd ~/IPL2026_Fantasy

STEP 2: COPY YOUR FILES
──────────────────────
Copy all these files to ~/IPL2026_Fantasy/:
  
  ✓ IPL_2026_Fantasy_Team_Builder.xlsx
  ✓ IPL_2026_Players_Comprehensive.xlsx
  ✓ IPL_2026_My11Circle_Fantasy_Rules.xlsx
  ✓ team_builder_basic.py
  ✓ team_builder_advanced.py
  ✓ CLAUDE_CODE_SETUP_GUIDE.md

STEP 3: INSTALL PYTHON LIBRARIES
────────────────────────────────
Run this command:

  pip install openpyxl pandas numpy

STEP 4: RUN YOUR FIRST SCRIPT
─────────────────────────────
Run this command:

  python team_builder_basic.py

Expected output:
  ✓ Team Builder initialized
  ✓ Loaded 400+ players
  ✓ Teams built successfully!

STEP 5: USING WITH CLAUDE CODE
───────────────────────────────
First, get API key from https://console.anthropic.com/

Then set it:
  export ANTHROPIC_API_KEY="your-key-here"

Then run Claude Code:
  claude-code "Run the Python script ~/IPL2026_Fantasy/team_builder_basic.py"

TROUBLESHOOTING:

Problem: "ModuleNotFoundError: No module named 'openpyxl'"
Solution: pip install openpyxl pandas numpy

Problem: "FileNotFoundError: IPL_2026_Players_Comprehensive.xlsx"
Solution: Copy all .xlsx files to ~/IPL2026_Fantasy/ folder

Problem: "ANTHROPIC_API_KEY not set"
Solution: export ANTHROPIC_API_KEY="your-key-from-console.anthropic.com"
